Redis

2023/12/7 NoSql 中间件 redis

  • 2.String字符串
  • 3.List列表

1.Key键

命令 描述
keys* 查看所有key
del 删除key
exists 判断key是否存在
type 查看key的类型
expire 设置过期时间
ttl 查看过期时间
select 切换数据库
dbsize 查看当前数据库的key数量
flushdb 清空当前库
flushall 通杀所有库

2.String字符串

命令 描述
set [NX / XX] 添加键值对 [ 不存在 / 存在 ]
get 获取键值对
incr / decr 将key中的存储的数字+1 / -1
incrby/decrby 将key存储的数字减少或增加指定的数（步长）
mset 同时设置一个或者多个key-value 键值对
mget 同时获取一个或者多个value
setnx 设置一个key-value 键值对，当key不存在才执行
getrange 获取值的范围，类似java的substring
setrange 覆盖指定位置字符串(结束位置长度就是value的长度)
getset 获取key中value的同时改变value的值

3.List列表

命令 描述
lpush/rpush 从左边/右边插入一个或者多个值
lpop/rpop 从左边/右边删除一个或者多个值
rpoplpush 从 列表的右边取出一个值，插到列表的左边
lrange 按照索引下标获取元素（从左到右) (0,-1取出所有值)
llen 获取列表长度
linsert 在的后面插入插入值
lrem 从左边删除n个value(从左到右)
lset 将列表key下表为index的值替换成value

4.set集合

命令 描述
sadd 将一个或者多个member元素加入到集合key中
smembers 取出该集合所有值
sismembe 判断集合key是否含有该value值，有1，无0
scard 返回该集合元素个数
srem 删除集合中某个元素
spop 随机从该集合中取出一个值并删除
srandmember 随机从该集合取出n个值，不会从该集合删除
smove 把集合中一个值移动到另一个集合
sinter 返回两个集合交集元素
sunion 返回两个集合并集元素
sdiff 返回两个集合的差集元素

5.Hash哈希

命令 描述
hset 给集合中的键值复制
hget 从集合取出value
hmset 批量设置key的value值
hexists 查看哈希表key中给定域field是否存在
hkeys 列出该hash集合所有field
hvals 列出该hash集合所有value
hincrby 为哈希表key中的域field值加上增量1 -1
hsetnx 将哈希表key中的域field值设置为value，当且仅当域field不存在 (不存在就添加，存在不添加)

6.Zset有序集合

命令 描述
zadd 将一个或者多个member元素已经score值加入有序序集key中
zrange 返回有续集key中，下标在 之间的元素（带WITHSCORES可以让分数一起和值返回到结果集）
zrangebyscore 返回有序集key中，所有score值介于min和max之间（包括min或max）的成员。有序集成员按score值从小到大排列
zrevrangebyscore 返回有序集key中，所有score值介于min和max之间（包括min或max）的成员。有序集成员按score值从大到小排列
zincrby 为元素score加上增量
zrem 删除该集合下指定元素
zcount 统计该集合，分数区间内的元素个数
zrank 返回该值在集合中的排名，从0开始
